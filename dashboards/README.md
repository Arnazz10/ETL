# Dashboard Guide

Use the CSV files generated in `output/dashboard/` as direct sources for Power BI or Tableau.

## Suggested visuals

- `titles_dashboard.csv`: master dataset for slicers, filters, and detail tables
- `top_10_genres_dashboard.csv`: bar chart of most common genres
- `content_added_per_year_dashboard.csv`: line chart of content growth over time
- `movies_vs_tv_ratio_dashboard.csv`: donut chart for content mix
- `top_10_countries_dashboard.csv`: map or bar chart by country
- `rating_distribution_dashboard.csv`: bar chart for rating mix

## Power BI / Tableau setup

1. Open Power BI Desktop or Tableau.
2. Connect to the CSV files inside `output/dashboard/`.
3. Build one summary page and one drilldown page.
4. Publish screenshots in the repository once the dashboard is built.
